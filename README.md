# Obsidian Tab Switcher

A plugin allows you switch currently opened tabs in Obsidian.

# How To Use

1. Open the command palette and type `Tab Switcher: Open Switcher For Tabs` to open the tab switcher.
2. OR you can set a hotkey for the command `Tab Switcher: Open Switcher For Tabs` in the hotkey settings.

- You can press key to search tab name;
- You can press hotkey to choose which tab to switch to.
- You can press `Enter` to switch to the first tab you searched.
